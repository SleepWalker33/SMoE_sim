# SMoE_sim

Baseline code for SMoE / MoE simulation experiments.

## Overview

This repository contains simulation code for running baseline experiments related to sparse mixture-of-experts style models.

## Environment and Run

Please first create the environment from the provided file, and then run the code:

```bash
conda env create -f environment.yml
conda activate mmlab
Or install dependencies with pip:
pip install -r requirements.txt
python run_reps.py
